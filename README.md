# Rent Pro-Rata Calculator 

A lightweight and responsive web calculator to quickly compute **pro-rata rent values** for **up to three rental contracts**, including individual amounts and the **total combined rent**, across **28, 29, 30, or 31-day months**.

Built with **HTML, CSS, and vanilla JavaScript**, focused on clarity, speed, and real-world rental use cases.

---

## Features

- Supports **28, 29, 30, and 31-day months**
- Calculates **pro-rata rent** based on occupied days
- Handles **up to 3 rental contracts simultaneously**
-  Displays **individual pro-rata values** and **total sum**
- Instant calculation (no backend, no dependencies)
- Fully responsive and mobile-friendly
- Clean, intuitive UI for quick operational use

---

## Use Cases

- Rental and real estate operations
- Lease transitions (move-in / move-out mid-month)
- Property management calculations
- Financial validation and reconciliation
- Quick checks without spreadsheets

## Tech Stack

- HTML5
- CSS3
